# IPL Data Analysis using Excel

## Overview

This project analyzes Indian Premier League (IPL) match data using Microsoft Excel.
The objective of this project is to explore team performance, player statistics, and match insights through data analysis.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Data Visualization
* Excel Dashboards

## Project Analysis

In this project, I analyzed:

* Team performance in IPL matches
* Player batting and bowling statistics
* Match results and trends
* Venue-based performance

## Dashboards

* Team Performance Dashboard
* Player Performance Dashboard
